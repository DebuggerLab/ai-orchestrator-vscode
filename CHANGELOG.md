# Changelog

All notable changes to the AI Orchestrator extension will be documented in this file.

## [1.0.0] - 2026-02-18

### Added
- Initial release of AI Orchestrator VS Code Extension
- Multi-model orchestration supporting OpenAI, Anthropic, Gemini, and Moonshot
- Intelligent task routing based on task type detection
- Sidebar chat panel with modern UI
- Task history with re-run and export capabilities
- Secure API key storage using VS Code Secrets API
- Context menu integration for selected code
- Status bar showing orchestration status
- Command palette commands for all operations
- Configurable model preferences
- Real-time progress indicators
- Consolidated output with syntax highlighting
- Markdown export functionality

### Technical
- TypeScript implementation with full type safety
- Webview-based UI with CSP security
- Support for VS Code 1.85.0+
